# External Resources Used

## Frameworks and Build Tools

- React: UI framework.
- Vite: frontend build tool and local development server.

## 3D and Rendering Libraries

- Three.js: WebGL rendering, geometry, materials, lights, shadows, and textures.
- React Three Fiber: React renderer for Three.js.
- Drei: helper components including `OrbitControls`, `Stars`, `Environment`, `Lightformer`, `Loader`, and `useProgress`.
- React Three Postprocessing: bloom and vignette effects.
- Postprocessing: post-processing effect support.

## Development Tools

- Oxlint: JavaScript/React linting.
- npm: package management.

## Assets

- Saturn surface texture: procedurally generated in code.
- Ring texture: procedurally generated in code.
- Nebula haze and sun glow: procedurally generated in code.
- Favicon and icons: local SVG assets in `/assets`.
- No downloaded third-party image, model, or video assets are required for the application.
